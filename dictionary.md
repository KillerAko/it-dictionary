# Терміни веб-розробки

## HTML (HyperText Markup Language)
Мова розмітки, яка використовується для створення структури веб-сторінок.

## CSS (Cascading Style Sheets)
Мова стилів, яка використовується для оформлення веб-сторінок.

## JavaScript
Мова програмування, що використовується для додавання інтерактивності на веб-сторінках.

## API (Application Programming Interface)
Інтерфейс, який дозволяє різним програмам взаємодіяти між собою.

## Responsive Design
Підхід до створення веб-дизайну, що адаптується під різні розміри екранів.

## Сервер
Комп’ютер або програмне забезпечення, яке обробляє запити клієнтів і надсилає їм відповіді, наприклад, веб-сторінки.

## Браузер
Програма, яка використовується для доступу до веб-сторінок через інтернет. Наприклад, Google Chrome, Firefox, Safari.

## HTTP (HyperText Transfer Protocol)
Протокол передачі даних, який використовується для обміну інформацією між клієнтом і сервером.

## DOM (Document Object Model)
Структура, яка представляє HTML-документ у вигляді дерева, що дозволяє JavaScript взаємодіяти з елементами сторінки.
